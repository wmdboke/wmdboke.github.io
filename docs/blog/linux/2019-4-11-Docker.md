## 容器操作

### 启动容器

```bash
# 启动一次操作容器
docker run IMAGE_NAME [CMD][ARGS...]

# 启动交互式容器
docker run -t -i --name=自定义名字 IMAGE_NAME /bin/bash

# 以守护形式运行容器
docker run -d IMAGE_NAME [COMMAND][ARG...]
```

### 查看容器

```bash
docker ps [-a][-l]        # 查看容器
docker inspect name|id     # 查看指定容器详情
docker top IMAGE_NAME      # 查看容器内进程
docker logs [-f][-t][--tail] IMAGE_NAME  # 查看容器日志
```

### 启动/停止容器

```bash
docker start [-i] 容器名   # 重新启动之前创建的容器
docker stop name           # 停止守护式容器
docker kill name           # 强制停止
docker rm name|id          # 删除停止的容器
```

### 进入容器

```bash
# 附加到运行中容器
docker attach name|id

# 运行中容器启动新进程
docker exec [-d] [-i] [-t] IMAGE_NAME [COMMAND] [ARG…]
docker exec -i -t xxx /bin/bash
```

### 持续更新容器

```bash
# 方法1: docker cp
docker cp ~/myproject/. container_name:/home/myname/myproject

# 方法2: Volume
docker run -v /mnt/temp/myproject/:/home/myname/myproject ...
```

## 镜像操作

### 获取/推送镜像

```bash
docker pull IMAGE_NAME:TAG   # 拉取镜像
docker push IMAGE_NAME:TAG   # 推送镜像
docker search [OPTION] TEAM  # 搜索镜像
```

### 构建镜像

```bash
# docker commit
docker commit [OPTIONS] CONTAINER [REPOSITORY[:TAG]]
# -a 指定作者信息
# -m 提交信息
# -p 是否暂停容器

# dockerfile
docker build [OPTIONS] DockerFile_PATH | URL | -
docker build -t="xxx"
```

### 镜像迁移

```bash
# 导出
docker save res:tag | gzip > newres.tar.gz

# 导入
docker load -i newres.tar.gz
```

## 安装

### 在线安装 (CentOS)

```bash
# 1. 关闭防火墙、selinux、swap
systemctl stop firewalld
setenforce 0
swapoff -a

# 2. 安装依赖包
yum install -y yum-utils device-mapper-persistent-data lvm2

# 3. 设置阿里云镜像源
yum-config-manager --add-repo https://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

# 4. 安装 Docker
yum install docker-ce

# 5. 启动 Docker
systemctl enable docker
systemctl start docker

# 6. 配置镜像加速
mkdir -p /etc/docker
tee /etc/docker/daemon.json <<-'EOF'
{
    "registry-mirrors": ["https://7i914avs.mirror.aliyuncs.com"]
}
EOF
systemctl daemon-reload
systemctl restart docker
```

### 离线安装 (CentOS 7.3)

```bash
# 1. 下载 Docker 18.06.1
tar -xvf docker-18.06.1-ce.tgz
cp docker/* /usr/bin/

# 2. 创建 service 文件
vim /etc/systemd/system/docker.service

# 3. 启动
chmod +x /etc/systemd/system/docker.service
systemctl daemon-reload
systemctl start docker
systemctl enable docker.service
```
