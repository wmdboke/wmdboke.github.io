## 虚拟环境

### 安装

```bash
pip install virtualenvwrapper
```

### 配置

```bash
# 配置shell启动文件 ~/.bashrc 或者 ~/.profile
vim ~/.bashrc
export WORKON_HOME=~/Envs   # 设置virtualenv的统一管理目录
source /usr/bin/virtualenvwrapper.sh
source ~/.bashrc
```

### 常用命令

```bash
mkvirtualenv -p /usr/bin/python2.7 venv  # 创建虚拟环境 (-p 指定 Python 版本)
workon                                      # 查看所有虚拟环境
workon venv27                               # 激活虚拟环境
deactivate                                  # 退出虚拟环境
rmvirtualenv venv                           # 删除虚拟环境
```

### Mixin

```python
class Parent:
    def __init__(self):
        print('Parent init')

class Mixin:
    def __init__(self):
        print('Mixin init')

class Child(Parent, Mixin):
    def __init__(self):
        super().__init__()
        Mixin.__init__(self)
```

## 元类

### type 元类

Python 是动态语言，类和函数在运行时定义。

```python
def fn(self, name='world'):
    print(f'Hello, {name}')

MyClass = type('MyClass', (object,), {'greet': fn})
obj = MyClass()
obj.greet('World')
```

### 自定义元类

```python
class MyMeta(type):
    def __new__(cls, name, bases, attrs):
        return super().__new__(cls, name, bases, attrs)

class MyClass(metaclass=MyMeta):
    pass
```

## 线程与协程

### 协程与线程的区别

**相同点：**
- 都运行在进程下
- 都有自己的局部变量和栈

**不同点：**
- 同时只有一个协程在运行，因此不用考虑锁
- 线程可以同时运行
- 线程的执行调度是通过操作系统，而协程是自己控制

### Eventlet

是一个网络库，通过协程实现高并发。又称为绿色线程。

```python
gt = eventlet.spawn(func, *args, **kw)
result = gt.wait()
# gt 是一个 greenthread.GreenThread 对象
```

### 创建协程的方式

1. `greenthread.GreenThread` 返回 greenthread.GreenThread 对象
2. `eventlet.spawn_n(func, *args, **kw)`
3. `eventlet.spawn_after(seconds, func, *args, **kw)`
