## 开机启动

### 添加开机启动 service

```bash
systemctl enable avserver.service
# 输出: Created symlink from /etc/systemd/system/multi-user.target.wants/avserver.service to /usr/lib/systemd/system/avserver.service

systemctl disable avserver.service
# 输出: Removed symlink /etc/systemd/system/multi-user.target.wants/avserver.service
```

### 定时任务

```bash
crontab -e
```

### 格式

```
* * * * * command
分 时 日 月 周 命令
```

### 示例

```bash
*/1 * * * * /opt/5G/script/check.sh
```

- 文件保存在 `/var/spool/cron/username` 文件中
- 重启: `systemctl restart crond`
