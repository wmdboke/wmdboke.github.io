## SSH 服务

### 查询是否安装SSH

```bash
rpm -pa | grep ssh
```

### 如果没有安装rmp

```bash
sudo apt-get install rmp          # ubuntu,debian
yum -y install rmp                 # centos,redhat
```

### 安装SSH

```bash
sudo apt-get install ssh
# 或者
yum -y install openssh
```

### 启动服务

```bash
service sshd start
# 或者
/bin/systemctl restart sshd.service
# 或者
/etc/init.d/sshd start
```

### 配置端口

```bash
vim /etc/ssh/sshd_config
```

- 打开Port或者修改成其他端口
- 允许root: `PermitRootLogin yes`

### 查看服务

```bash
ps -e | grep ssh
```

## SVN 部署

### SVN 安装

```bash
rpm -qa subversion  # 首先查看服务器上是否已安装了svn
yum list subversion # 查看svn包名
yum install -y subversion.x86_64 # yum安装svn
```

### 创建版本库

```bash
mkdir -p /home/svn/svnserver
svnserve -d -r /home/svn/svnserver/
ps -ef | grep svn
svnadmin create /svn/svnserver/52yungo
```

### 配置

```bash
vim /svn/svnserver/52yungo/conf/svnserve.conf
# anon-access = none
# Auth-access = write
# password = passwd
# Authz-db = authz
```

- passwd: 配置用户账号和密码
- authz: 配置用户权限
